# Creation Engine Raytracing
A SKSE plugin for hardware raytracing on compatible Creation Engine games. 

| Game | Support | Requirements |
|----------|----------|----------|
| Skyrim SE/AE  | Yes (in development)  | [Community Shaders](https://github.com/community-shaders/skyrim-community-shaders/) |
| Fallout 4  | Planned (needs RE) |

### Features
- Pathtracing
- Raytraced indirect illumination (diffuse and specular)

#### Supports
- Subsurface Scattering
- Hair BSDF
- ReSTIR GI
- Stable Planes/Primary Surface Replacement
